# SmoothMaps
A Minecraft mod to provide smooth lighting and performance improvements for framed maps, paintings and rails.

Respects your smooth lighting setting and doesn't cause a big performance hit.

### Features
- Fast smooth lighting for framed maps
- Fast smooth lighting for paintings
- Enables smooth block lighting for rails
- Improved painting performance by removing interior faces
- Improved server-side map rendering performance (up to 60% faster)
- Reduced GPU bandwidth overhead for map rendering (partial texture updates)
